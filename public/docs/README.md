## 功能介绍

*   **思维导图视图**: 提供多种思维导图（Markmap, 思绪视图）来可视化装修流程。
*   **节点详情**: 点击导图节点可查看详细信息和相关图片。
*   **工具下载**: 内置装修账单等实用工具，点击节点下载。

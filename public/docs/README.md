#### 1. 交互式思维导图

点击任意节点，展开下级内容。准备阶段下藏着收房验房、装修公司挑选、设计步骤等；施工阶段里细分水电、泥木、油漆……层级清晰，想查哪里点哪里。

每个知识点不只是标题。点击叶节点备注图标，弹出详细说明+实景图片+验收要点。比如"水电"节点，你能看到水电易漏点位、水电材料验收标准、施工规范、验收步骤、常见避坑提醒等。

## 功能介绍

*   **思维导图视图**: 提供多种思维导图（Markmap, 思绪视图）来可视化装修流程。
*   **节点详情**: 点击导图节点可查看详细信息和相关图片。
*   **文档查阅**: 内置文档浏览器，方便查阅装修知识。
*   **用户登录**: 支持用户登录、信息记录与个性化功能。
*   **数据同步**: 思绪视图支持从服务器获取最新流程数据。
